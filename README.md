A web-based Pharmacy Management System developed for **Satota Pharmacy & Departmental Store**, Uttara, Dhaka.

## Built With
- PHP 8.2
- MySQL 8.0
- Bootstrap 5
- XAMPP (Apache + MySQL)
- JavaScript

## Features
- Role-based login (Admin, Pharmacist, Cashier, Inventory)
- Medicine inventory with low stock and expiry alerts
- Sales & billing with automated invoice generation
- Multiple payment methods (Cash, Card, bKash, Nagad)
- Prescription management
- Supplier and customer management
- Reports and analytics with date filter
- Real-time dashboard

## How to Run
1. Install XAMPP and start Apache + MySQL
2. Import `pharmacy_db.sql` into phpMyAdmin
3. Copy the `pharmacy` folder to `C:/xampp/htdocs/`
4. Open browser → `http://localhost/pharmacy/`
5. Login: `admin@pharmacy.com` / `password`
